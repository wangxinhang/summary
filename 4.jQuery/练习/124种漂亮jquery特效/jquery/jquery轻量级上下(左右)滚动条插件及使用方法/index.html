现在网上充斥着大量的jquery滚动条代码，但往往含有多个插件，使用起来比较麻烦，且很难解决一个页面多次使用的问题


使用方法

1、复制代码到你需要的地方，且依次引入jquery库以及tinyscrollbar插件即可。

2、然后在网页底部直接按照此代码调用库中的效果即可 $('#scrollbar').tinyscrollbar(); 默认为竖向滚动条效果。如果需要多次使用，只需改变外侧的ID号，并修改css样式即可
